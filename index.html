Nossos projetos:




Repositório GitHub


Já que eu não podia ter um sistema de chamados profissional, decidi fazer o meu próprio! O SysChamados é um sistema que auxilia em chamados internos da empresa quando um usuário tem algum problema relacionado à minha área. E, com ele, consigo ter um controle de produtividade e de serviços já feitos através dos procedimentos salvos no banco de dados. O sistema é todo feito com html, css, PHP e um pouco de JavaScript. Acesse o meu repositório no GitHub e baixe os manuais de usuário e técnico do sistema para mais informações.
